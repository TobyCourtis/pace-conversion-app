# Running Pace Conversion App

An Ionic test!!

Currently, supports pace to speed e.g 5min/km -> 12km/h

<br>

## Requirements

- node v20.0.0

## Frontend

#### Start

`ionic serve`

#### Fronted client manual run

`npx tsx BackendClient.ts`

## Backend

`cd backend && npm start`


## Building frontend

`npm run build`

`npx cap sync`

`npx cap open ios`
